🎨 Design — Desktop

Langkah membuat desain di Figma desktop; setiap langkah disertai foto agar mudah diikuti.

1. Buka figma.com atau aplikasi Figma Desktop.
2. Tekan untuk membuat Frame.
3. Tekan T untuk menambahkan teks.
4. Gunakan panel kanan untuk mengatur font, warna, ukuran, dan line height.
5. Tambahkan efek di bagian Effects.
6. Atur warna melalui menu Fill.
7. Klik Export untuk menyimpan desain.
