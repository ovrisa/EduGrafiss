📈 Infografis — Mobile

📱 Cara membuat infografis di Canva mobile. Ikuti langkah mulai dari buka aplikasi sampai mengunduh hasil.

1. Buka Aplikasi Canva
Login akun.
2. Cari “Infografis” lalu pilih Template
Tap template sesuai kebutuhan.
3. Edit Judul dan Teks
Tap teks → edit isi, font, warna dari menu bawah.
4. Ganti Ikon
Tekan + → Elements untuk pilih ikon tema, atau pilih Uploads dari galeri.
5. Rapikan Tata Letak
Geser objek dengan jari → gunakan Position untuk atur layer.
6. Download Infografis
Tap Share → Download → pilih PNG atau PDF.
