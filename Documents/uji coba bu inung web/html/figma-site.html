💻 Figma Site — Desktop

Langkah-langkah membuat desain site di Figma (desktop/browser). Ikuti setiap langkah untuk memulai dari file kosong sampai mengekspor desain.

1. Klik New Site File
Mulai file baru di browser atau aplikasi desktop.
2. Tekan lalu pilih ukuran Desktop
3. Gunakan Rectangle (R) dan Text (T)
4. Atur warna melalui Fill di panel kanan
5. Gunakan Layout (Shift + A) agar jarak otomatis rapi
6. Gunakan Group (Ctrl + G) untuk mengelompokkan layer
7. Masukkan gambar dengan Place Image (Shift + Ctrl + K)
8. Klik Export untuk menyimpan desain
