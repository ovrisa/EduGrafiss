📄 Poster — Desktop

Halaman tutorial desktop untuk membuat poster di Canva. Ikuti langkah-langkah di bawah beserta gambar contoh.

1. Buka Canva & Login
Masuk ke www.canva.com → login dengan Google/email.
2. Klik "Create a Design"
Pilih Poster atau ketik "Poster" di kolom pencarian.
3. Pilih Template
Template ada di panel kiri → klik untuk menggunakannya.
4. Edit Teks
Klik teks → ubah font, ukuran, warna di toolbar atas.
5. Ganti Gambar
Klik menu Photos (kiri) atau Uploads untuk upload sendiri. Drag & drop ke desain.
6. Tambah Elemen
Klik menu Elements → pilih ikon, shape, garis, dll.
7. Atur Layer & Posisi
Klik objek → pilih Position (kanan atas) untuk atur maju/mundur.
8. Tambah Logo
Upload logo → atur ukuran & posisi.
Download
Klik Share (kanan atas) → Download → pilih PNG / PDF Print.
