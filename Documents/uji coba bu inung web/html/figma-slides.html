📽️ Slides — Desktop

Langkah membuat presentasi di Figma desktop; setiap langkah dilengkapi foto agar mudah diikuti.

1. Klik New Design File atau pilih template “Presentation/Slides”
Mulai file baru di browser atau aplikasi desktop.
2. Buka Figma lalu klik New Design File atau cari template “Presentation/Slides” di Figma Community.
3. Tekan F lalu pilih ukuran 1920×1080 px atau preset Presentation.
4. Klik teks untuk mengedit, tekan T untuk menambah teks baru.
5. Gunakan Layout agar tata letak rapi.
6. Buat Komponen untuk header, footer, atau nomor slide supaya konsisten.
7. Atur warna di menu Fill dan atur font di panel kanan.
8. Duplikat slide dengan Ctrl/Cmd + D.
9. Klik Export lalu pilih PNG atau PDF.
