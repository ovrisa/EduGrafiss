📋 FigJam — Desktop

Halaman tutorial desktop untuk membuat brainstorming, flowchart, dan kolaborasi di FigJam. Ikuti langkah-langkah di bawah beserta gambar contoh.

1. Klik New FigJam File
Mulai file baru di browser atau aplikasi desktop.
2. Gunakan Sticky Notes untuk menulis ide
3. Ubah warna dan ukuran sticky sesuai kebutuhan
4. Tambahkan Shapes seperti kotak, lingkaran, panah
5. Tekan T untuk menambahkan teks tambahan
6. Atur ukuran dan style teks di panel kanan.
7. Gunakan menu manage widgets untuk menambahkan setiker atau emoji
8. Klik Share untuk kolaborasi real time
