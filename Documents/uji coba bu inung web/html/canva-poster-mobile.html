📄 Poster — Mobile

Halaman tutorial mobile untuk membuat poster dengan aplikasi Canva. Ikuti langkah dan lihat contoh gambar.

1. Buka Aplikasi Canva
Login akun.
2. Tekan Tombol "+" (Bawah Tengah)
Pilih Poster.
3. Pilih Template
Scroll template → tap untuk pakai.
4. Edit Teks
Tap teks → muncul menu edit di bawah layar (Font, Color, Effects).
5. Ganti Gambar
Tap gambar → pilih Replace → ambil dari galeri atau Photos.
6. Tambah Elemen
Tekan tombol + → pilih Elements, Text, Uploads, dll.
7. Atur Layer
Tap objek → pilih Position → atur Forward/Backward.
8. Download
Tap Share (pojok kanan atas) → Download → pilih format.
